# Suggested conference usage

For an oral presentation or poster, this repository can be linked as a reproducible companion workflow. Recommended material to show:

1. The conceptual workflow from BUSCO outputs to ranked marker panels.
2. The PCA ranking plot and explanation of PC1 as a marker-quality axis.
3. The saturation curve used to justify the optimal panel size.
4. Downstream comparison of fixed panels and full datasets, when available.

Recommended wording:

> We developed a reproducible BUSCO-derived marker-prioritization workflow that ranks orthologous loci using taxon occupancy, alignment quality and phylogenetic informativeness metrics. The ranked markers are then assembled into flexible panels and empirically evaluated across different phylogenetic depths.
